# smogon-player-study

Corpus and analysis for the Smogon Player Study Program — elite-player dossiers built from tournament replay logs. `harvest/` pulls replays from their native `.log` endpoints; `corpus/<player>/` holds immutable game logs plus a manifest; `discovery/` holds per-player game inventories from archival research; `analysis/<player>/` holds quantitative fingerprints and dossiers.
